## 📑 功能特点

- 自动展示本周课表信息
- 可以直接下载
- 响应式布局

## 🚀 快速开始

1. 填写后端地址 `./src/components/CnameData.jsx`

   ```jsx
   export const ApiURL = 'http://1.117.154.114:20000'
   ```

2. 📦 安装并启动应用

   ```bash
   cd frontend
   npm i
   npm run dev
   ```
